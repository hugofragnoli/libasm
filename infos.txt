Instructions de flux et de pile : 

je : "Jump if equal"
push / pop : Ajoute ou retire une valeur du haut de la pile (push rdi save ladresse de ta chaine avant que tu nappelles une autre fonction
qui pourrait ecraser ce registre).
call : Appelle une fonctione externe.
leave : Prepare la sortie de la fonction (raccourci pour restaurer la pile).
ret : Sort de la fonction. Reprend l exec la ou la fonctiona ete appeleee. Le res doit ETRE DANS RAX.

Instructions arithmetiques et Logiques :

mov : copie une valeur d un endroit a un autre. Ex : mov r8, rax ne deplace pas la donnee, ca la duplique.
movzx : (move with Zero extend) : Copie un petit element dans un grand registre en remplissant le reste de zeros.
    Indispensable pour lire un char (1 octet) et pouvoir faire des calculs dessus sans garder des residus
    de vieilles données dans le registre.
xor : Operation logique. Ex : xor rax, rax est la maniere la plus rapide et opti de mettre un registre a zero.
inc : Incremente de 1 : utilisé pour passer au char suivant dans une chaine.
imul : multiplication signee -> Mulitplie le registre par la valeur donnee. Ooperation couteuse en temps process 
    par rapport a laddition.
neg : Transforme un nb positif en neg -> utilise le complement a deux. Cest ce qui permet de gerer le - dans atoi base.


Instructions de test et de comparaison.

cmp : compare une valeur a celle d un registre. -> Soustrait virtuelllemt la valeur B de la valeur A.
    Ex : si A vaut 10 et B 9 le res du cmp sera 1 donc A plus grand que B 

test : compare une valeur a elle-meme  -> si le registre est a "zero", le drapeau ZERO FLAG sactive.
    On sen sert pour verif les pointeurs NULL.


Les Jumps

jmp : saute toujours -> sans condition
jz : jump if zero -> saute si res precedent = 0
je : jump if equal -> saute si les deux valeurs du cmp sont identiques
jne : jump if not equal -> saute si les deux valeurs du cmp sont differentes.
jl : jump if less -> saute si A < B.
jg : jump if greater -> saute si A > B.
jbe : Jump if below or equal -> Saute si A <= B
ja : Jump if above -> Saute si A > B. Utilise pour detecter les overflow.
